# カレンダー「観戦」ステータス表示修正 議事録

**日付**: 2026-06-30

---

## 質問

jleague_watching_calendarの開発を再開します。
議事録から過去の状況は確認してください。

勝敗の登録を「観戦」としていた場合に、ホーム画面のカレンダー上から確認できる結果には「未定」と表示されるようになっていました。
こちらについても他同様登録されているステータス「観戦」が表示されるようにしてください

---

## 回答

*議事録と参照ファイルを確認*

*match_calendar_widget.dart・match_result.dart を確認*

*home_screen.dart を確認*

原因が分かりました。`home_screen.dart:147` の `_getResultText` メソッドに `watch` ケースが無いため、デフォルトの「未定」が返されていました。

`MatchOutcome.watch` は `isFinished`（`outcome != MatchOutcome.tbd`）が `true` になるため、ダイアログの「結果: スコア (...)」の分岐には入るものの、`_getResultText` の switch 文に `'watch'` ケースが存在せずデフォルトの「未定」が表示されていました。

*home_screen.dart の `_getResultText` に `case 'watch': return '観戦';` を追加*

修正後の `_getResultText`:

```dart
String _getResultText(String? result) {
  switch (result?.toLowerCase()) {
    case 'win':
      return '勝利';
    case 'lose':
      return '敗北';
    case 'draw':
      return '引き分け';
    case 'watch':
      return '観戦';
    default:
      return '未定';
  }
}
```

これでカレンダーから試合をタップした際に「結果: スコア (観戦)」と正しく表示されます。
